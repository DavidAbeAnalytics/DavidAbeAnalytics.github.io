About

Building data infrastructure
that scales.

I'm a Cloud Data Engineer and Analytics Engineer who got into this field wanting to understand how data moves: how it's pulled from source systems, stored, transformed, and made accessible to the teams that need it. That curiosity is still what drives my work today.

My experience covers the full data stack on GCP: ingesting raw data from source systems into Cloud Storage, transforming it inside BigQuery using dbt, and surfacing it through Power BI dashboards that business teams can act on. I also work with Apache Kafka for real-time streaming pipelines, bringing event-driven data into the same warehouse layer. I care about data quality and clean modelling as much as the pipeline itself. A well-tested dbt model matters as much as the code that feeds it.
